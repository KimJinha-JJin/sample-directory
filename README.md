# 🏎️ Max Verstappen Fan Site - #1
> **"Simply, simply lovely!"** - AI와 함께한 즐거운 'Vibe Coding' 프로젝트

## 🌟 Project Overview
F1의 전설, **막스 베르스타펜(Max Verstappen)**을 위한 헌정 팬 사이트입니다. 
단순히 정보를 나열하는 것을 넘어, 레드불 레이싱의 역동적인 에너지와 속도감을 웹 기술로 구현하는 데 집중했습니다. 

## ✨ Key Features
* **Dynamic Loading Screen**: 막스의 엔트리 넘버 '1'을 강조한 강렬한 오프닝 연출
* **Custom F1 Cursor**: 사이트 전체에서 F1 머신 모양의 전용 커서 사용
* **Live Stats Counter**: 월드 챔피언십 횟수와 승수를 생동감 있게 보여주는 애니메이션
* **Responsive Timeline**: 2015년 데뷔부터 현재까지의 커리어 하이라이트 정리
* **Interactive Quote Slider**: 막스의 상징적인 명언들을 확인할 수 있는 슬라이더

## 🛠 Tech Stack
* **Frontend**: HTML5, CSS3, JavaScript (Vanilla JS)
* **Vibe Tech**: AI(Claude Opus)와의 협업을 통한 자연어 기반 코드 생성 (Vibe Coding)

## 🎨 Design Concept
* **Color Palette**: Red Bull Racing의 시그니처 컬러(Dark Blue, Red, Yellow) 활용
* **Inspiration**: 평소 취미인 PPT 제작 감각을 녹여내어 시각적인 균형과 임팩트를 강조했습니다.

## 🚀 How to See
1. 이 레포지토리를 클론하거나 파일을 다운로드합니다.
2. `index.html` 파일을 브라우저에서 실행하면 바로 확인할 수 있습니다!
